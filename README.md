# IG Video Controls

| # | 功能 | 說明 |
|---|---|---|
| 1 | 可拖曳進度條 | 仿 YouTube 風格的進度條，顯示緩衝區段，滑鼠懸停可預覽時間戳記 |
| 2 | 音量滑桿 | 滑鼠移入影片時顯示橫向滑桿，可即時調整音量 0–100% |
| 3 | 全螢幕按鈕 | 一鍵切換全螢幕，圖示自動隨進入／退出狀態變換 |
| 4 | 自動取消靜音 | 控制列注入時自動取消靜音並將音量設為 100% |
| 5 | M 鍵快捷鍵 | 按 `M` 鍵切換目前最活躍影片的靜音狀態，在輸入框中輸入時不觸發 |
| 6 | 靜音提示浮層 | 切換靜音時顯示短暫淡入淡出的提示文字，提供清楚的視覺回饋 |
| 7 | Instagram SPA 支援 | 使用 `MutationObserver` 與導航輪詢，偵測頁面切換時動態載入的影片並自動注入控制列 |

| # | Feature | Description |
|---|---|---|
| 1 | Seekable Progress Bar | Draggable YouTube-style progress bar with buffered-range indicator and hover tooltip showing timestamp |
| 2 | Volume Slider | Horizontal volume slider (0–100%) that appears on mouse hover over the video |
| 3 | Fullscreen Button | One-click fullscreen toggle; icon switches between enter/exit states automatically |
| 4 | Auto Unmute | Automatically unmutes videos and sets volume to 100% when controls are injected |
| 5 | M Key Shortcut | Press `M` to mute/unmute the most visible playing video; disabled when typing in input fields |
| 6 | Mute Toast Notification | Brief overlay fades in/out when mute state changes, providing clear visual feedback |
| 7 | Instagram SPA Support | Uses `MutationObserver` and navigation polling to detect dynamically loaded videos across page transitions |
